## Identified Patch Opportunities

1. **Drawbridge CIDER adapter** — no existing CIDER middleware for
   Drawbridge. Would allow `M-x cider-connect` to a Drawbridge
   endpoint directly. High value for the Clojure community.

2. **XTDB client improvements** — the compat write layer in futon1a
   has workarounds (props not in standard entity shape, non-UUID ID
   resolution). Some of these could be upstreamed.

3. **Datascript pull pattern for refs** — the `[*]` pull doesn't
   include ref entity details. Documented during WebArxana build.

4. **http-kit WebSocket improvements** — the WebArxana WS layer
   revealed edge cases in connection management.
